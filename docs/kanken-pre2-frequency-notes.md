# 漢検 準2級 頻出語メモ（作問の語選定の参考）

計画48の作問で「未出題の配当字を頻度順に埋める／頻出の定番語を厚く出す」ための参考資料。
2026-06-13 に複数の独立した無料学習サイトを Web 横断調査し、**2ソース以上で一致した語**を中心に集約した。

## この資料の位置づけ・制約

- 出典は学習サイト・ブログの「頻出」記述の**ソース間一致**。過去問コーパスそのものの統計ではない（協会が著作権を持ち機械分析できる形では入手不可）。
- 市販問題集の頻出度A/B/Cランクや問題文・配列は**取り込まない**（著作権。計画39の非スコープ）。ここにあるのは「複数ソースが共通して頻出と挙げた語そのもの」の事実集約。
- 読み・意味・例文は作問時に自前で簡潔に書き起こす（リストの丸写しはしない）。
- 「⭐」= 2ソース以上で一致（確度高・優先採用）。無印 = 単一ソースだが定番と明記されていた語。
- 【未】= アプリで未出題の準2級配当字（213字）を含む。読みカバレッジ・シリーズの最優先ターゲット。

## 配点と目標（計画48の根拠の再掲）

## 6. 部首（暗記のみ・質の修繕用）

複数ソースで「間違えやすい頻出」：戻(戸)／扇(戸)／扉(戸)／煩(火)／臭(自・3ソース)／塁(土)／累(糸)／尉(寸)／恭(⺗)／慕(⺗)／衷(衣)／衰(衣)／賓(貝)／丙(一)／妥(女)

パターン：①字全体が部首（斉・竜・缶・麻・辛・舌）が毎回1問程度②候補が複数で迷う字（戻=戸か大、塁=土か田）③部首が埋もれる字（甚=甘、美=羊、朱=木）④「迷ったら口」（喪・呉・嗣）。

## 7. 熟語の構成（5パターンの例）

- ア 同義：帰還／謙譲／珠玉／疾患／享受／治療
- イ 対義：硬軟／繁閑／粗密／親疎／栄辱／緩急
- ウ 修飾：傑作／駄作／墨汁／硝煙／漸進／公僕
- エ 下が目的語：迎賓／殉教／懐郷／赴任／克己（「～ヲ/～ニ」で訓読できる）
- オ 打ち消し：未納／無窮／非・未・不・無 で始まる語

## 8. 誤字訂正（同音・形似の定番／量産しない）

定番ペア（誤→正）：模策→模索／示佐→示唆／折辱→雪辱／付及→普及／適示→指摘／転付→添付／名柄→銘柄／選択使→選択肢／元固→頑固／退掘→退屈／溝読→購読／写音→遮音。
二大要因：同音異字（普/付・索/策・購/溝）と形似字（摘/適・肢/使・把/破）。作問コストが高く配点10点なので量は増やさない。
