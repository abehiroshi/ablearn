# フッタナビゲーション（アイコン案C）

- 状態: 未着手
- 触るファイル範囲: `src/App.tsx`・`src/components/`（FooterNav 新規）・`src/styles.css`
- 引き継ぎメモ: デザイン仕様は `assets/ablearn-icons.png` の**案C（ミニマル・丸形ベース）**。カラーパレットも同画像の指定に従う

## 目的

5画面（ホーム・学習=Library・テスト=模擬テスト・復習・成績=Stats）への常設ナビゲーションを
画面下部に固定し、どの画面からも1タップで移動できるようにする。

## 内容

- 案Cのデザインを仕様として**SVGで起こす**（PNG切り出しにしない）。
  選択中/非選択の色替え・バッジ表示をCSSで制御でき、解像度非依存のため
- 5項目: ホーム / 学習 / テスト / 復習 / 成績
- 復習アイコンに復習対象の件数バッジを表示
- 既存の画面遷移導線と二重になる箇所（ホームのボタン等）は整理する
- タブレット（05/08のレイアウト）とスマホの両方で崩れないこと

## 非スコープと破綻条件

- 画面構成・画面数の変更はしない（導線の置き換えのみ）
- クイズ解答中・模擬テスト中はフッタを隠す（誤タップで離脱しないため）→
  途中離脱の導線が欲しくなったら明示的な「やめる」ボタンで対応

## 受け入れ条件

- 全画面でフッタが表示され、1タップで相互に移動できる
- 選択中の画面がアイコンの状態でわかる
- タブレット縦横・スマホ縦で崩れない
- 模擬テスト中に誤って離脱できない
