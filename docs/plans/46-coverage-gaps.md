## 内容

- **数学・新単元「箱ひげ図とデータの活用」**: 概念 `math-shibunni`（四分位数・四分位範囲の計算）と
  `math-hakohige`（箱ひげ図の読みとり・複数データの比較）。学年末タグ。四分位は計算系としてフル変種
- **数学・一次関数の増強**: 新概念 `math-ichiji-glaph`（グラフをかく・読む）・`math-ichiji-heniki`（変域）・
  `math-ichiji-kousaten`（交点と連立方程式、prereq: renritsu）・`math-ichiji-riyou`（利用、d3中心）。
  既存2概念とあわせ単元6概念へ。グラフは座標読み取りの choice／値の input で出題（作図そのものは扱わない）
- **理科・電気の後半**: `sci-denryoku`（電力・電力量・発熱量。計算系フル変種）・`sci-seidenki`（静電気・電子・真空放電）・
  `sci-jiba`（電流がつくる磁界・磁界から受ける力）・`sci-denjiyudo`（電磁誘導と発電）。prereq はオーム系に接続
- **理科・天気の補完**: `sci-kumo`（露点と雲のでき方。湿度計算との役割分担を整理）・
  `sci-nihon-kishou`（気団・大気の動きと日本の四季）。既存 `sci-zensen-tenki` と前提鎖でつなぐ
- **理科・化学の補完**: `sci-hatsunetsu`（発熱・吸熱反応と身近な例）。知識寄り d1-2 中心の小概念
- **社会・中部地方**: `soc-chiri-chubu`（東海・中央高地・北陸の3区分と産業の対比）＋用語カード1セット。
  既存の西日本/東日本セットの体裁に合わせる
- **国語・敬語**: `jpn-keigo-shurui`（尊敬・謙譲・丁寧の識別）を概念ファミリーで。教科書の扱いを確認のうえ
  必要なら言い換え（〜れる/お〜になる等）まで
- **国語・短歌**: 形式（音数・句切れ）と表現技法の小セット。漢詩セットの体裁に合わせる
