# 出典つきセット（学校テスト由来の問題と再テスト導線）

- 状態: 未着手
- 触るファイル範囲: `src/types.ts`・`src/screens/LibraryScreen.tsx`・`scripts/validate-content.mjs`
- 引き継ぎメモ: 運用フローは `assets/TASKS.md` のS系ルールを参照

## 目的

学校で実際に使われた小テスト・定期テスト・ワーク由来の問題セット（S系タスクで生成）に
出典を持たせ、「あのテストのやり直し」が選べるようにする。

## 内容

- `SetMeta` に `origin?: string` を追加（例: `"2026-06-10 数学小テスト"`）。任意フィールド・既存互換
- Library で origin つきセットに出典バッジを表示し、由来がわかるようにする
- 解き直しは既存機能のまま: 通常挑戦は `setRecords` がベスト・履歴を記録、
  本番形式でやりたければ模擬テスト（06）の範囲にそのセットを指定する
- 検証スクリプトに origin の形式チェック（文字列・長すぎない）を追加

## 非スコープと破綻条件

- 元ネタ写真の表示・添付はしない（public リポジトリに置かないため）→
  「元の問題と見比べたい」が出たらドライブのリンクを origin に併記する形を検討
- 出典別の専用画面は作らない → 出典つきセットが増えて一覧性が悪くなったら Library にフィルタを足す

## 受け入れ条件

- origin つきセットが Library で見分けられる
- origin なしの既存セットの表示が変わらない
- 検証スクリプトが origin の不正な型を検出する
