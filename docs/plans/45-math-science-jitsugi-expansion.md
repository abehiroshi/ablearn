# 数学・理科の変種増強と実技4教科の拡充

- 状態: 未着手
- 触るファイル範囲: `public/content/chugaku/{math,science,music,art,pe,tech-home}/`・`public/content/chugaku/concepts.json`・`public/content/chugaku/index.json`・`scripts/validate-content.mjs`（概念別チェックのエラー昇格のみ）。アプリコードには触れない
- 引き継ぎメモ: 高水準化プログラムの最終便。計画42〜44の完了後に着手し、
  完了時に計画41で警告扱いにした概念別チェック（変種5以上等）をエラーへ昇格して締める。
  ベースライン: 数学97問(concept 65%)・理科77問(concept 47%)・実技 各10問1セット

## 目的

数学・理科は仕組みが効いている分、変種の絶対数が足りない（間隔反復と再戦が回ると
「答えを覚えた」状態になる）。実技4教科は各10問で定期テスト対策の量に達していない。

## 内容

- **数学**: 全習熟概念を計画41基準（変種5以上・input可能2以上・stage0/1/2）へ。
  手続き概念はフル変種（自己説明・誤答修正・faded）。concept 未付与の35%を解消
- **理科**: 同上。計算系（湿度・オームの法則・電力量・化学反応の質量比)は数学と同じ
  フル変種、知識系は flashcard との役割分担を整理して concept率 85% へ
- **実技4教科**: 各教科3セット以上へ拡充（音楽=楽典＋鑑賞曲・美術=色彩＋技法・
  保体=体力テスト＋救急/生活習慣・技家=食生活＋情報/製図 など定期テストの定番領域）。
  実技は知識中心のため flashcard＋choice 主体でよく、concept は無理に付けない
  （計画41の concept率基準は主要5教科のみに適用、実技は除外とする）
- **締め**: validate:content の概念別チェックを警告→エラーに昇格（計画41で予約した移行の完了）

## 非スコープと破綻条件

- 数学の作図・証明の記述採点は扱わない（証明は穴埋め choice / 並べ替え order で出す）
- 破綻条件: 変種を一気に増やすと既存の習熟度データと不整合になることはない（概念単位で
  足すだけ）が、**既存問題の id 変更は履歴を切る**ので追補のみとする（計画41と同じ規約）

## 受け入れ条件

- chugaku の全習熟概念が概念別チェック（変種5以上・input可能2以上・stage展開）を
  エラーモードで通過する
- 実技4教科が各3セット以上
- 数学・理科の concept 付与率 85% 以上
