# 諸地域を教科書の章（地方）ごとに分割し、九州を増補

- 状態: 完了（2026-06-14）
- 出典: オーナー指示（実テストの範囲表 = 九州地方まで）。学校テスト写真に基づく割り込み（S系相当）。
  会話で①表記は親の章名を含める「地理（日本の諸地域　◯◯地方）」②諸地域は教科書の章ごとに全分割、を確定
- 触るファイル範囲: `public/content/chugaku/index.json`・`public/content/chugaku/concepts.json`・
  `public/content/chugaku/social/chiri-*.json`。アプリコードには触れない

## 背景（確認結果）

範囲表が「九州地方まで」なのに、アプリの諸地域は西日本（九州・中国四国・近畿）と東日本（関東・東北・
北海道）を1ユニットに結合していた。範囲選択はセット単位（計画35）なので「九州だけ」を選べず、
九州の問題も実質5問しかなかった。教科書（教育出版）の章は地方ごとなので、単元名も章と一致させたい。

## 内容

- 諸地域を教科書の章＝地方ごとに7ユニットへ分割。単元名は親章を含め「地理（日本の諸地域　◯◯地方）」
  （歴史と区別する `地理（…）` ラッパーは維持）。第2章は「地理（日本の特色と地域区分）」に改称
- 地方別セット（各セット1概念）と用語カードを用意し、品質ゲート（変種5・input可2・stage0/1/2・
  explanation・2段ヒント）を満たすよう既存問題を地方別 concept に付け替え＋増補:
  - 九州地方 `soc-chiri-kyushu`（15問）: 水俣病・環境モデル都市・筑紫平野/二毛作・地熱・
    シリコンアイランド・自動車工業・さんご礁・エコツーリズム・畜産 を追加
  - 中国・四国地方 `soc-chiri-chugoku-shikoku`（7問）／近畿地方 `soc-chiri-kinki`（6問）
  - 関東地方 `soc-chiri-kanto`（6問）／東北地方 `soc-chiri-tohoku`（6問）／
    北海道地方 `soc-chiri-hokkaido`（6問）
- terms タグ: 九州=`2学期中間`（範囲表どおり）、中国四国・近畿=`2学期期末`、中部・関東・東北・北海道=`学年末`
- `concepts.json`: 旧 `soc-chiri-nishinihon`/`soc-chiri-kanto-hokkaido` を地方別概念に分割
  （chugoku-shikoku / kinki / kanto / hokkaido）。tohoku・kyushu は set を地方セットへ付け替え
- 旧結合ファイル（chiri-shochiiki・chiri-higashinihon と各 cards）は削除

## 効果（受け入れ）

- 範囲選択で地方を1つずつチェックできる（九州だけ＝範囲表と一致）＝粒度の問題を解消
- 単元名が教科書の「日本の諸地域　◯◯地方」と一致
- `validate:content` エラーゼロ・全172テスト通過。社会の難易度3比率20%・concept率98%

## 破綻条件・引き継ぎ

- **履歴**: 地方分割でセットIDが変わるため、旧結合セットでの解答履歴は切れる（分割の不可避なコスト）。
  以後は id 不変で増補する（計画41/46の規約に復帰）
- **terms タグは暫定**: 「中間=九州まで」からの推定。学校の範囲表で確定したら index.json で調整
  （編集UIは非スコープ）。中国四国・近畿と中部以降の所属テストがズレたらタグ修正で吸収
