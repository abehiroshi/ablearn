# 手書き計算余白（クイズ画面のタブレット対応込み）

- 状態: 未着手
- 触るファイル範囲: `src/screens/QuizScreen.tsx`・`src/components/`・`src/styles.css`
- 引き継ぎメモ: なし

## 目的

数学・理科の計算問題で、画面上に計算用紙の代わりとなる手書きスペースを提供する。
主端末はタブレット＋スタイラス。

## 内容

- クイズ画面に Canvas の手書きエリア。ペン・消しゴム・全消去のみの最小ツール
- タブレット横画面では「左に問題、右に余白」の2カラム。スマホではボタンで開くオーバーレイ
- Pointer Events で実装し、ペン入力中はタッチを無視する（パームリジェクション）
- 問題を切り替えたらクリア。保存しない
- 全問題形式で利用可（教科や形式での出し分けはしない。使わなければ開かないだけ)

## 非スコープと破綻条件

- 手書き認識・採点はしない → 手書きの答えを正誤判定したくなった時点で破綻。別計画として起こす
- 書いた内容の保存・見直しはしない → 「テスト前に自分の計算過程を見返したい」が出てきたら破綻

## 受け入れ条件

- スタイラスで遅延なく書ける（タブレット実機で確認）
- ペンで書いている最中に手のひらが触れても誤描画・誤スクロールしない
- スマホでも開閉でき、問題への解答操作を妨げない
