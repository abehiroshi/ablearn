# 社会の概念体系化と増強

- 状態: 未着手
- 触るファイル範囲: `public/content/chugaku/social/`・`public/content/chugaku/concepts.json`・`public/content/chugaku/index.json`。アプリコードには触れない
- 引き継ぎメモ: 計画41の完了後・計画42と同型の作業（42の進め方を踏襲すると速い）。
  ベースライン: 65問・concept率 0%・ヒント率 57%・answers率 54%・難易度3 8問

## 目的

社会も concept ゼロで習熟エンジンが働いていない。暗記教科こそ間隔反復の効果が大きい。

## 内容

- **概念ファミリー化**: 「用語の暗記」でなく「区別・因果」を概念単位にする
  （例: 江戸の三大改革の区別・日本の気候区分と雨温図・開国条約の不平等点の説明・
  日清/日露戦争の講和条約の対比）。地理と歴史で前提宣言（例: 雨温図 ← 気候区分）
- **全演習問題に concept を付与**し、計画41基準の変種構成へ拡張
- **難易度3を資料読み取り・記述型で増強**: 雨温図/地形図/年表/史料の読み取りを input で
  （挑戦束の弾。記述は answers の表記ゆれ列挙で受理）
- **一問一答系は flashcard に整理**: 用語→説明・説明→用語の両方向

## 非スコープと破綻条件

- 画像が必要な資料問題（地図・グラフ画像）は、画像アセットの用意が必要なら T系タスクに
  切り出す（テキストで再現できる資料＝表・年表から先に作る）
- 破綻条件: 歴史の記述 answers は言い回しの幅が大きく、列挙で受理しきれない問題は
  choice に倒す（無理に input にすると理不尽になる）

## 受け入れ条件

- 社会の演習問題の concept 付与率 85% 以上・計画41の教科別基準をすべて満たす
- 各単元にセット2以上
- 地理・歴史それぞれで間隔反復（定着確認）が実際に回る（手元確認）
