# 学習の可視化（カレンダー・得意不得意・成長）

- 状態: 未着手
- 触るファイル範囲: `src/screens/StatsScreen.tsx`・`src/lib/`
- 引き継ぎメモ: 成長グラフは [04](04-answer-history.md) の記録が貯まるほど描ける期間が伸びる。実装を急ぐより記録開始（04）を先行させる判断済み

## 目的

自分の積み重ねと伸びが見えることをモチベーションにつなげる。

## 内容（作る順）

1. **学習カレンダー（草表示）**: 日々の解答量をヒートマップ表示。`dailyLog` に必要データが既にあり最安
2. **得意不得意レーダー**: 教科別・単元別の正答率。`questionStats` から計算できる。円グラフではなくレーダー（凹みが直感的に見える）
3. **成長グラフ**: 単元別正答率の時系列折れ線（04の日次集計から）。「不得意だった単元が上がっていく」が見える
4. **ヒントなし正解率**: [07](07-difficulty-and-hints.md) のヒント記録が貯まったら、正答率より正確な習熟指標として追加

## 非スコープと破綻条件

- グラフライブラリは入れず SVG 直書きの方針 → 凝ったインタラクション（ズーム・ツールチップ多用）が欲しくなったら再検討
- 他人との比較・共有はしない（1人用）

## 受け入れ条件

- データが少ない初期状態でも空表示が破綻しない（「まだデータがないよ」を Abler が言う等）
- タブレット・スマホ両方で見やすい
