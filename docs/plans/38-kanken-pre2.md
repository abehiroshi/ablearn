## 目的

漢検準2級の対策を、中学教科書とは独立した世界 `/ablearn/kanken/` で提供する。
コンテンツは **準2級のみ** から始める。他の級は本人の必要が出た時点で追加する
（オーナー指示 2026-06-12: まず準2級だけ・出題形式は本番の全分野）。

## コレクション（器は計画22で完成済み・手順はREADME記載）

- URL: `/ablearn/kanken/`、`collections.json` に `id: kanken` を追加
- PWA名（仮）: **「漢検問題集」**。オーナーが別名を指定したら上書きする
- 進捗キー: `ablearn:kanken:v1`（計画22の仕組みのまま。chugaku と完全独立）

## コンテンツ構造（本番の全分野）

subject は「準2級」の1教科構成、unit を本番の出題分野に対応させる。

| 分野（本番の配点） | 形式 |
|---|---|
| 読み（30問×1点） | input（ひらがな入力） |
| 部首（10問×1点） | choice |
| 熟語の構成（10問×2点） | choice（ア〜オ分類） |
| 四字熟語・書き（10問×2点） | flashcard（手書き→自己採点） |
| 四字熟語・意味（5問×2点） | choice |
| 対義語・類義語（10問×2点） | flashcard（手書き→自己採点） |
| 同音・同訓異字（10問×2点） | flashcard（手書き→自己採点） |
| 誤字訂正（5問×2点） | input または choice |
| 漢字と送りがな（5問×2点） | flashcard（手書き→自己採点） |

- 書き取り（25問×2点）は「四字熟語・書き」等と同じ flashcard 方式の独立 unit とする
- 手書きはタブレット＋スタイラス前提。出題→余白に書く→模範解答を見て自己判定
- 初期投入は **全分野で最低1セット（1セット10問前後）を一巡** → 以降セットを順次増強。
  準2級配当漢字（約328字）のカバー率を index 側のセット名で追えるようにする
- 問題は配当漢字・出題形式（事実）に基づくオリジナル生成。協会の過去問・問題文は転載しない。
  生成時の検証観点（誤字訂正の正答一意性・四字熟語の実在確認）を `docs/content-generation.md` に追記する

## 非スコープと破綻条件

- 他の級（4級・3級・2級）は対象外。追加するときは unit 追加ではなく subject 追加で対応する。
  級ごとに進捗世界を分けたい要望が出たら、コレクション分割を含めて構造を見直す
- 本番形式の模擬テスト（60分・200点の再現）はやらない → 必要になったら模擬テスト（06）の
  コレクション対応として別計画を起こす
- 手書き文字の自動採点はしない（自己採点方式）。とめ・はね・画数の判定要望が出ても自動化しない
- 級配当の読み（高校レンジの音訓）を網羅しきれない場合は、セット名で範囲を明示し
  「未収録範囲がある」ことを隠さない

## 受け入れ条件

- `/ablearn/kanken/` がランディングから遷移でき、PWAとして単独インストールできる
- 全9分野（＋書き取り）に最低1セットがあり、choice / input / flashcard が一巡動作する
- コンテンツ検証（CI）が kanken を含めて通る（エントリ・マニフェスト漏れ検出は計画22の validator）
- chugaku の進捗・解答履歴に影響がない（localStorage 分離の既存テストが通る）
