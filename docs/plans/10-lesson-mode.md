# 単元学習モード（レッスン）

- 状態: 未着手
- 触るファイル範囲: `src/types.ts`・`src/screens/`・`public/content/math/`（試行1単元）
- 引き継ぎメモ: なし

## 目的

いきなり演習ではなく、教科書のように順を追って「短い解説 → 易しい問題」を1つずつ進めて理解するモードを追加する。アプリを「演習ドリル」から「教材」に広げる。

## 内容

- セットに `kind: "lesson"` を導入。中身は「解説カード」と既存形式の問題が混ざったステップ列を順に進む
- 採点プレッシャーを下げる: 間違えてもその場で解説、XPは小さく（+2程度）
- Library 上で演習セットと見分けがつく表示
- コンテンツはまず数学の進行中単元1本だけ作って体験を確かめる

## 非スコープと破綻条件

- 全教科・全単元への展開はこの計画に含めない → 1本試して良ければ、コンテンツ生成パイプライン（レッスン・ヒント・難易度をまとめて生成）を別計画として起こす
- 動画・図版は扱わない（テキストのみ）→ 図がないと説明できない単元（図形・グラフ）で破綻する。その時に画像対応を検討

## 受け入れ条件

- レッスンを最初から最後まで進めると完了が記録される
- 既存の演習セットの動作・進捗に影響がない
